Accessibility rules (WCAG AA)
  • Amber, orange, cyan & cream all pass AA on Void/Panel — safe for body text.
  • Use Magenta only at ≥16px or bold (5.5:1). Never for tiny text.
  • On gradient/amber fills, text is always Void #140F0A, never white.
  • Muted meta uses Sand #B49C6A (4.8:1) — don't go dimmer for essential info.
